## Scope

 <p style="font-size: 1.115rem;">
  Responsive front-end recreation of Nike's landing page, built with HTML, CSS and vanilla JavaScript.</p></li>

 <p style="font-size: 1.115rem;">
  Focus has been on replicating the visual layout, spacing and responsiveness of the original page across different screen sizes. JavaScript is used to add interactivity to the hero carousel, including auto-play, manual navigation and dot indicators.</p></li>

 <p style="font-size: 1.115rem;">
  A few css features for showing and expanding hidden list items found in the footer section in smaller viewports are left out because of time constraints.</p></li>

 <p style="font-size: 1.115rem;">
  Placeholder images are used instead of Nike's original assets.</p></li>
